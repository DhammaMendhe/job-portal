# Job Board App 🚀

A full-stack Job Board application built with the MERN stack and TypeScript.

## Tech Stack

**Frontend**
- React + TypeScript
- Redux Toolkit
- React Router
- Axios
- Tailwind CSS

**Backend**
- Node.js + Express + TypeScript
- MongoDB + Mongoose
- JWT Authentication
- Nodemailer

## Features

- 🔐 User authentication (Register/Login with JWT)
- 📝 Post job listings (Employers)
- 🔍 Browse and search jobs (Job Seekers)
- 📩 Apply for jobs
- 📊 Dashboard for managing posts and applications
- 📧 Email notifications on registration and application

## Getting Started

### Backend
```bash
cd backend
npm install
# Add your .env file (see .env.example)
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
